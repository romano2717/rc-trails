Time Attack Challenge Scoring (30% time, 70% deductions)

Rank Player Time Time (s) Deductions TimeScore DeductionScore Points Remove
Formula per event (based on current entrants):
TimeScore = (Tmax - Ti)/(Tmax - Tmin) (or 1 if all times equal)
DeductionScore = (Dmax - Di)/(Dmax - Dmin) (or 1 if all deductions equal)
Points = 100 * (0.30*TimeScore + 0.70*DeductionScore)
Tie-breakers: higher points → lower deductions → lower time.
